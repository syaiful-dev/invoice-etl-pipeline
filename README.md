# Invoice ETL Pipeline

## Overview

This project demonstrates a simple ETL process that imports invoice data from CSV into SQL Server using Pentaho Data Integration (PDI).

---

## Features

- CSV Input
- Data Validation
- Duplicate Detection
- Database Insert
- Logging
- Error Handling

---

## Technology

- Pentaho PDI
- SQL Server
- CSV

---

## Architecture

CSV

↓

Pentaho

↓

Validation

↓

Success Log
